# Documentation

![feature-sliced-banner](/img/banner.jpg)

**Feature-Sliced Design** (FSD) es una metodología de arquitectura para organizar aplicaciones front-end. En pocas palabras, es un compilado de reglas y convenciones sobre la organización del código. La motivación principal de FSD es hacer el proyecto más fácil de entender y mantener estructurado de cara a los (siempre cambiantes) requerimientos del negocio.
